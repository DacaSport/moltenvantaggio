# Daca Sport C.A. - Landing Page Balón Molten Vantaggio

Landing page móvil lista para publicar. Está diseñada para vender el Balón Molten Vantaggio y enviar pedidos directo a WhatsApp con mensaje automático según la selección del cliente.

## Incluye

- Inicio corregido y ordenado para teléfono.
- Branding Daca Sport con paleta rojo, azul, negro y blanco.
- Producto dedicado: Balón Molten Vantaggio.
- Variantes: rojo, azul, número 4, número 5 y fútbol sala.
- Precios automáticos por cantidad y método de pago.
- Compra directa por WhatsApp.
- Envío gratis por Zoom o Tealca.
- Retiro en tienda física.
- Garantía de válvula.
- Ubicación: C.C. Girarma, local #2, Guanare Edo. Portuguesa.

## Publicación

Sube todo el contenido de esta carpeta al hosting. El archivo principal es `index.html`.

El WhatsApp está configurado en `assets/js/app.js` con el número usado en esta versión.
